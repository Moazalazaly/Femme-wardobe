Femme Wardrobe

Femme Wardrobe is an e-commerce website specializing in women's wear and accessories, offering a stylish and user-friendly shopping experience.

🚀 Features
- Parallax Background Effect: Enhances the visual experience with a smooth scrolling effect.  
- Smooth Page Transitions: Implements smooth scrolling and animated navigation between pages.  
- Dynamic Product Listing: Fetches real-time product data from an external API, ensuring up-to-date inventory.  
- Responsive Design: Fully optimized for mobile, tablet, and desktop screens.  
- Lazy Loading for Images: Improves page speed by loading images only when needed.  
- Native Web Technologies: Built using 'pure' HTML, CSS, and JavaScript without any frameworks.
